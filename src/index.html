Проекты

Утилиты Bash для работы с текстом
Разработана серия скриптов Bash, облегчающих манипуляции с текстовыми файлами. Задачи включали сортировку текста по разным параметрам, поиск и замену подстрок, разделение и объединение текстовых файлов.

Собственная реализация математической библиотеки math.h
Проект включал в себя реализацию основных математических функций из библиотеки math.h. Особое внимание уделялось точности и производительности. Результаты функций были проверены на соответствие стандартной библиотеке.

Реализация собственной библиотеки string.h на языке С с дополнениями
Проект был направлен на создание библиотеки для работы со строками, аналогичной string.h, но с дополнительными функциями. Разработанные функции включали разделение строки по разделителю, замену подстроки, переворачивание строки и др.

Реализация собственной библиотеки s21_decimal.h
В этом проекте была разработана библиотека для работы с десятичными числами. Задача включала в себя создание функций для арифметических операций, округления и форматирования чисел.

Установка и обновление системы Linux
Проект включал в себя установку и обновление операционной системы Linux, а также настройку рабочего окружения и установку необходимого программного обеспечения. Задачи включали решение проблем совместимости, оптимизацию производительности и установку драйверов.
